TP01- This project provides a comprehensive introduction to the Pandas library, one of the most widely used data analysis tools in Python. The presentation covers the fundamentals of Pandas, including its overview, core data structures (Series and DataFrame), basic functionalities, input/output operations, data indexing and selection techniques, visualization methods, and its relationship with NumPy. In addition, a hands-on tutorial demonstrates practical data analysis using a real-world weather dataset obtained from Kaggle. Through step-by-step examples, the project illustrates data cleaning, transformation, exploration, statistical analysis, and visualization using Pandas DataFrames. The objective is to develop a strong understanding of data manipulation and analysis techniques that are commonly applied in data science, machine learning, and business analytics.

TP02- This project introduces Scikit-learn, a popular Python library for machine learning and predictive analytics.
It covers core concepts including supervised learning, unsupervised learning, model selection, evaluation, and visualization techniques.
The project demonstrates machine learning algorithms such as linear regression, support vector machines, clustering, and Gaussian mixture models.
A hands-on tutorial using a Kaggle dataset showcases data preprocessing, model training, performance evaluation, and result visualization.

TP03- This project introduces PyTorch, a powerful open-source deep learning framework used for building and training neural networks.
It covers fundamental concepts such as tensors, datasets and data loading, data transformations, neural networks, and automatic differentiation using torch.autograd.
The project demonstrates how to design, train, and evaluate deep learning models for real-world applications.
A hands-on tutorial using an MNIST classification or Natural Language Processing dataset showcases data preprocessing, model development, training, and performance analysis.
